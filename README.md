# Gerenciador-de-Projetos
Desenvolvimento de um sistema web full-stack para gerenciar arquivos XLSX com cabeçalho padronizado, incluindo pipeline de tratamento automático dos dados, persistência em banco de dados e comunicação entre módulos via Flask. O sistema utiliza inteligência artificial para interpretar o conteúdo da planilha selecionada, permitindo consultas contextuais, busca otimizada e geração de respostas inteligentes com base apenas nos dados carregados pelo usuário.
